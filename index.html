            Browser notifications
            Forward to Slack / Teams / Discord

            Paste an Incoming Webhook URL from your own Slack/Teams/Discord. While you have BRS Community open in any tab, matching events are forwarded there — so your phone's Slack/Teams/Discord app rings you.

            📖 How to set up Slack (private channel for yourself)
            1. In your Slack workspace, create a private channel just for yourself (e.g. #brs-notifications) so only you see the forwarded messages.
            2. Open my.slack.com/services/new/incoming-webhook. Choose your private channel and click Add Incoming WebHooks integration.
            3. Copy the Webhook URL (starts with https://hooks.slack.com/services/…) and paste it below.
            4. Save, then click Test below to confirm a message lands in your private Slack channel.

            If your workspace blocks legacy Incoming WebHooks, ask your admin to install the Incoming WebHooks app from the App Directory and post in your channel — the URL ends up the same.

            📖 How to set up Microsoft Teams
            1. In Teams, pick (or create) a channel just for yourself.
            2. Click the channel's ••• menu → Connectors → search Incoming Webhook → Add → Configure.
            3. Give it a name (e.g. BRS Chat), upload an icon if you want, then click Create.
            4. Copy the URL and paste it below. Save, then Test.
            📖 How to set up Discord
            1. In Discord, create a private channel only you can see (Server Settings → Roles, or use a personal server).
            2. Channel Edit Channel → Integrations → Webhooks → New Webhook.
            3. Click Copy Webhook URL and paste it below. Save, then Test.
